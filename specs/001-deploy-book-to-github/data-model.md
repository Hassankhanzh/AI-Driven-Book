### Repository
- **name**: AI-Book
- **visibility**: Public
- **defaultBranch**: main
- **description**: Physical AI & Humanoid Robotics Spec-Kit Plus project with Docusaurus book

### Docusaurus Configuration
- **url**: https://hassankhanzh.github.io
- **baseUrl**: /AI-Book/
- **organizationName**: Hassankhanzh
- **projectName**: AI-Book
- **title**: AI-Driven Book

### Book Content
- **format**: Markdown (.md)
- **modules**: 4 (Modules 1-4)
- **structure**: Organized in docs/ directory with sidebar navigation
- **validationRules**: Must render correctly in Docusaurus build

### Spec-Kit Plus Artifacts
- **components**: /sp.constitution, /sp.specify, /sp.plan
- **purpose**: Spec-Driven Development tools and templates
- **location**: Root project directory and .specify/ subdirectory

### Deployment Assets
- **source**: Docusaurus build output
- **target**: GitHub Pages
- **branch**: gh-pages (auto-generated by Docusaurus deploy)
- **validation**: Build process completes without errors

## Relationships

- Repository **contains** Docusaurus Configuration
- Repository **contains** Book Content
- Repository **contains** Spec-Kit Plus Artifacts
- Docusaurus Configuration **uses** Book Content
- Deployment Assets **generated from** Docusaurus Configuration + Book Content

## State Transitions

1. **Local Development** → **Build Process** → **Production Build**
2. **Local Repository** → **Git Push** → **GitHub Repository**
3. **Docusaurus Build** → **GitHub Pages Deployment** → **Public Site**
